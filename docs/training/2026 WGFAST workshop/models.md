# Using a model

A tutorial on how to use echoSMs scattering models is included in the echoSMs [documentation](../../usage.md#running-a-model).

A JupyterHub notebook that covers the same material is at `notebooks/running models.ipynb`.

## Using datastore shapes in a model

The `notebooks/web shapes to TS.ipynb` notebook shows how to get several shapes, calculate their target strength, then plot the results.
