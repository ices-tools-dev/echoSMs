# EchoSMs anatomical data store schema

**Title:** EchoSMs anatomical data store schema

|                           |                                                                |
| ------------------------- | -------------------------------------------------------------- |
| **Type**                  | `object`                                                       |
| **Additional properties** | ![Not allowed](https://img.shields.io/badge/Not%20allowed-red) |

**Description:** This document is the JSON schema for the echoSMs anatomical data store. It provides a standardised structure for storing fish and plankton acoustic scattering model data.

| Property                                                       | Type                      | Title/Description                                                                                                                                                     |
| -------------------------------------------------------------- | ------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| + [dataset_id](#dataset_id )                                   | string                    | A unique identifier for this dataset, generated by the data storage system.                                                                                           |
| + [description](#description )                                 | string                    | A description of the dataset.                                                                                                                                         |
| + [anatomical_category](#anatomical_category )                 | enum (of string)          | The type of anatomy that this dataset contains.                                                                                                                       |
| + [anatomical_features](#anatomical_features )                 | array of enum (of string) | The type of anatomical features in this dataset.                                                                                                                      |
| + [date_first_added](#date_first_added )                       | string                    | The date when these metadata were first created, in [RFC9557](https://datatracker.ietf.org/doc/rfc9557) format.                                                       |
| + [date_last_modified](#date_last_modified )                   | string                    | The date when these metadata were last modified, in [RFC9557](https://datatracker.ietf.org/doc/rfc9557) format.                                                       |
| + [aphiaID](#aphiaID )                                         | integer                   | The WoRMS [aphiaID](https://www.marinespecies.org/aphia.php) of the species in this dataset.                                                                          |
| - [class](#class )                                             | string                    | Taxonomical hierarchy level of class that is consistent with WoRMS.                                                                                                   |
| - [order](#order )                                             | string                    | Taxonomical hierarchy level of order that is consistent with WoRMS.                                                                                                   |
| - [family](#family )                                           | string                    | Taxonomical hierarchy level of family that is consistent with WoRMS.                                                                                                  |
| - [genus](#genus )                                             | string                    | Taxonomical hierarchy level of genus that is consistent with WoRMS.                                                                                                   |
| - [species](#species )                                         | string                    | Taxonomical hierarchy level of species that is consistent with WoRMS.                                                                                                 |
| - [vernacular_name](#vernacular_name )                         | string                    | A vernacular name for the organism.                                                                                                                                   |
| - [reference](#reference )                                     | string                    | A literature reference for the dataset. If available, a DOI or URI is recommended.                                                                                    |
| - [activity_name](#activity_name )                             | string                    | Name of the data collection activity (if any), such as voyage code, project name, etc.                                                                                |
| - [location](#location )                                       | string                    | Name or description of the location where organism was collected.                                                                                                     |
| - [latitude](#latitude )                                       | number                    | Latitude of location where organism was collected (decimal degrees).                                                                                                  |
| - [longitude](#longitude )                                     | number                    | Longitude of location where organism was collected (decimal degrees).                                                                                                 |
| - [depth](#depth )                                             | number                    | Depth below surface of organism collection location (m).                                                                                                              |

| + [date_image](#date_image )                                   | string                    | The date when the dataset was imaged, in [RFC9557](https://datatracker.ietf.org/doc/rfc9557) format.                                                                  |
| - [investigators](#investigators )                             | array of string           | Names or identifying descriptions of persons or organisations that created this dataset.                                                                              |
| - [data_collection_description](#data_collection_description ) | string                    | Description of data collection details.                                                                                                                               |
| - [note](#note )                                               | string                    | Explanatory notes about the shape data.                                                                                                                               |
| + [imaging_method](#imaging_method )                           | enum (of string)          | Method used to obtain the raw image data.                                                                                                                             |
| + [shape_method](#shape_method )                               | enum (of string)          | Method used to obtain the specimen shape.                                                                                                                             |
| - [shape_method_processing](#shape_method_processing )         | string                    | Description of processing used to convert raw images into shapes, including any manipulation of image data (for example, smoothing, edge enhancement, rotation, etc). |

| - [mass_density_method](#mass_density_method )                 | enum (of string)          | Method used to obtain density in the feature.                                                                                                                         |
| + [shape_types](#shape_types )                                 | array of enum (of string) | Shape types in this dataset. Generated by the data store server                                                                                                       |
| + [dataset_size](#dataset_size )                               | number                    | Size of the entire dataset (including raw data) (MiB). Generated by the data store server.                                                                            |
| + [specimens](#specimens )                                     | array of object           | Specimen data.                                                                                                                                                        |
