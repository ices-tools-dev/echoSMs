# Resonance Kirchhoff model

This directory contains information about a new resonance-Kirchhoff scattering model (Chu, 2024).

No code is currently provided here for this model, however a description of the model is provided in the `Resonance & Kirchoff Model 03-21-2024.pdf` file (the source file, `Resonance & Kirchoff Model 03-21-2024.docx` is also in this directory).

Chu, 2024. Development and Application of Advanced Science and Technology in Fisheries Acoustics. Journal of the Marine Acoustics Society of Japan, Vol 51, No. 4, 117-149 (available in this directory in `Chu et al. 2024 JMASJ_51-4_117_2024 Fisheries Acoustics.pdf`)
